Projects

Most recent work

Parkinson Disease Detection

Built a machine learning model to predict Parkinson’s disease at early stages, even before medical diagnosis. The model uses a voice recording dataset to identify patterns related to the disease. This helps in faster and more affordable detection, saving time and reducing the need for costly medical tests. The goal is to support early treatment and improve the patient’s quality of life.

Retail Analysis

Created a detailed Retail Analysis Dashboard using Power BI and Python to help understand how a retail business is performing. I worked with large amounts of data from different sources, cleaned it, and combined it into one model. The dashboard shows important information like sales trends, top-selling products, customer buying patterns, and stock levels. I used Python to do some advanced data work and to add custom charts. This project helped identify which areas needed improvement and supported better decisions in sales and inventory planning. Tools I used include Power BI, Python (Pandas, NumPy, Matplotlib)

Automating Bill Of Materials

During my internship, I worked on automating the Bill of Materials (BOM) process for a manufacturing company that previously relied on Excel sheets and manual entry. Our task was to clean and preprocess large sets of BOM data, which often had inconsistencies and missing values. We analyzed the data to identify patterns in material usage and created interactive dashboards to help the team understand production and inventory trends more easily. Additionally, we developed a machine learning model to predict the required materials for different product types, aiming to reduce manual errors and speed up the planning process. This project helped improve accuracy, save time, and support smarter decision-making in the manufacturing workflow.

Visual Reconstruction from Brain Activity

Worked on a deep learning project titled "Visual Reconstruction from Brain Activity", which involved two main stages. In the first stage, we developed a model to classify brain tumors using MRI scan data, accurately predicting whether a person was affected and identifying the tumor type. In the second stage, for those affected, we focused on reconstructing the visual image they might be seeing based on their brain activity. This involved analyzing brain signals and generating corresponding visual outputs using a deep learning model. The project aimed to bridge neuroscience and AI, offering insights into medical diagnostics and potential future applications in brain-computer interfaces.
